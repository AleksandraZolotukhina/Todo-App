### Используемые библиотеки
- React
- TypeScript
- Mobx-State-Tree
- ANTD

### Запуск проекта
1. Склонируйте репозиторий к себе на компьютер
```
git clone https://github.com/AleksandraZolotukhina/Todo-App.git
```

2. Установите зависимости проекта
```
npm i
```

3. Запустите проект
```
npm run dev
```
